# GAN-pp
implemente Generative Adversarial Network with paddlepaddle

## Pre-requisites
1. PIL
2. opencv-python
3. paddlepaddle 1.8.4
4. numpy

## Install
```shell
pip install -r requirements.txt
```

## How to use it

1. train network
```shell
    python train.py
```
2. use network for inference
```shell
    python infer.py
```
